## goproxy-heroku

### Heroku 是一个支持多种编程语言的云平台即服务，goproxy-heroku 则是可部署在 Heroku 平台的 goproxy 服务。goproxy-heroku 使用的 WebSocket 代替原本的 sockets 作为底层传输。
